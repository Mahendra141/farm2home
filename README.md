# Farm2Home

Farm2Home is a full-stack e-commerce application built using React, Redux, Spring Boot, and MySQL.

## Features
- Product browsing & cart management (Redux)
- OTP-based delivery verification
- Order creation & tracking
- WhatsApp notification to owner on order placement
- Secure backend APIs with JWT authentication
- Admin-ready architecture for product & order management

## Tech Stack
- Frontend: React, Redux, CSS Modules
- Backend: Spring Boot, Spring Security, JWT
- Database: MySQL
